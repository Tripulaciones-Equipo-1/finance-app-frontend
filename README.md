## Run Locally

Clone the project

```bash
  git clone https://github.com/Tripulaciones-Equipo-1/finance-app-frontend.git
```

Go to the project directory

```bash
  cd finance-app-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
